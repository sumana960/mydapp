# Mydapp

Mon application décentralisée (dApp) sur le réseau Monad.

## Description
Cette dApp permet :
- De se connecter à MetaMask
- D’interagir avec mon token ERC20 sur Monad testnet
- De visualiser le solde du token

## Technologies
- HTML
- JavaScript
- ethers.js
- MetaMask
- Monad Testnet
